# Static JSON database (GitHub Pages)

GitHub Pages is **static hosting**, so “database” here means **JSON files committed to the repo** and loaded in the browser via `fetch()`.

## Files

- `scraped_content/db/site_db.json`
  - Full site metadata: sections + activities + indexes.
- `scraped_content/db/trac_nghiem.json`
  - Convenience subset: activities tagged `trac_nghiem`.
- `scraped_content/db/trac_nghiem_questions.json`
  - Full question bank extracted from the trắc nghiệm HTML pages (prompt/options/correct).

## Generate / refresh

Run locally (then commit the JSON outputs):

- `python generate_db.py`

## Notes

- `localPath` is relative to `scraped_content/` (e.g. `pages/Some_Page.html`).
- `availableLocal` tells you whether the file exists in the repo (useful because quizzes/resources may be skipped during scraping).

## Question extraction

`trac_nghiem_questions.json` is generated by parsing the HTML in `scraped_content/pages/*.html`.
It looks for:

- Question starts: `Câu N:`
- Options: `A.` / `B.` / `C.` / `D.`
- Correct option: the option label is underlined in HTML (e.g. `<u>C.</u>`)
